Enter file in which to save the key (/c/Users/.ssh/id_ed25519): Enter passphrase (empty for no passphrase): Enter same passphrase again: Your identification has been saved in /c/Users/.ssh/id_ed25519. Your public key has been saved in /c/Users/.ssh/id_ed25519.pub. $ clip < ~/.ssh/id_rsa.pub ~~~~~~~~~~~~~~~~~~~~~~~~ 然后贴到github网站的ssh keys中 **问题2：origin方式由https修改成ssl方式遇到问题** $ git remote add origin git@github.com:用户名/用户名.github.io.git fatal: remote origin already exists. $ git remote rm origin $ git remote add origin git@github.com:用户名/用户名.github.io.git $ git remote -v origin git@github.com:用户名/用户名.github.io.git (fetch) origin git@github.com:用户名/用户名.github.io.git (push) $ git pull $ git add --all $ git commit -m "test" $ git push -u origin main 访问https://用户名.github.io/ 成功！
